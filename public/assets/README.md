# Public Assets

This folder contains assets that are safe to ship with the open-source Whistle repository.

## Layout

- `brand/`: neutral Whistle-owned placeholder marks and illustrations.
- `data/`: Whistle-owned sample geography/catalog data used by local prototypes.
- `fonts/`: bundled font files plus their upstream license notice.

## License Notes

- Brand SVG files in `brand/` are original Whistle placeholder assets and are covered by the repository MIT License.
- `data/tamil-nadu-districts.geojson` is a hand-authored schematic sample grid for prototype heatmaps. It is covered by the repository MIT License and must not be treated as official boundary or GIS data.
- Noto Sans Tamil font files in `fonts/` remain under the SIL Open Font License, Version 1.1. See `fonts/NOTO-SANS-TAMIL-LICENSE`.

## Asset Boundary

Do not commit official government emblems, department seals, party marks, public-figure portraits, third-party photographs, or video-only demo media here unless their public redistribution rights are documented and approved.

For stakeholder videos or private demos, keep those assets in a local ignored `private-assets/` folder or a separate private asset repository. See `docs/whistle-private-demo-assets.md`.
